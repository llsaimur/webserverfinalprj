PHP children's games website, which includes the following features:

Creation of user accounts or registration (Sign-Up).

Real-time validation of the information entered by the user in the form for creating user accounts with AJAX.

Sign-In to an existing user account.

Logout of a connected user account (Sign-Out and Time-Out).

Changing the password for an existing user account.

Management of several levels of a question-and-answer game that follow one another.

Abandonment of a game in progress.

Display of the history of the results of all the games ended.

Manipulation of a MySQL database to save, select, and update data collected and generated by the previous features.

Custom display of features and interaction of additional interactivity and attraction features
